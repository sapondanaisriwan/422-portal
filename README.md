## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

*   **Styling:** CSS + Tailwind CSS (v4)
*   **UI Components & Icons:** Lucide React, SVG Brand Icons
*   **Layout Library:** `react-responsive-masonry`
